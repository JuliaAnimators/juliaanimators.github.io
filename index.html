About JuliaAnimators

JuliaAnimators is a collection of packages, resources, and fun crew of maintainers dedicated to bringing novel animation and visualization to the world of the Julia programming language.

Currently, JuliaAnimators has several different supported packages but is built around the central package, Javis.jl. Check out our JuliaCon 2021 talk introducing Javis to learn more!

Packages Available

Here is a list of currently supported packages in JuliaAnimators:

PackageDescription
Javis.jlJulia Animations and Visualizations
JavisNB.jlNotebook integrations for Javis.jl
JavisViewerNative viewing tool for Javis written in GTK
JGraphs.jlSupport package to animate networks using Javis
